JACUZZI
CSS font, mille loomist on inspireerinud Yusugomori CSS SANS.
JACUZZI tähed on justkui pealtvaates mullivannid, kus viibib üks või enam mullivannitajat.
JACUZZI sobib ühesõnaliste mõtete edasiandmiseks. Kujuta ette, et mõnuled mullivannis ja keegi küsib: "Kuidas läheb?" ja sina vastad ...

KASUTAMINE
SAMM 1: JACUZZI kasutamiseks on vaja allalaadida siit jacuzzi.css.
SAMM 2: Paiguta jacuzzi.css file siht kodulehe /css kausta.
SAMM 3: Seejärel tuleb linkida jacuzzi.css stylesheet oma html-iga lisades "head" tag'ide vahele rida "link rel="stylesheet" href="css/jacuzzi.css"" ja võite nautida mullivanni mõnusid.
